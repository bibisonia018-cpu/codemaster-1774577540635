# نظام المتاجر المتعددة مع إشعارات تيليجرام

تطبيق احترافي يسمح للتجار بإنشاء متاجر خاصة وإضافة منتجاتهم، وتلقي إشعارات الطلبات فوراً عبر تطبيق تيليجرام.

## كيفية الإعداد:
1. **Firebase**: أنشئ مشروعاً جديداً وفعل `Authentication` و `Firestore`.
2. **Telegram Bot**:
   - تواصل مع `@BotFather` لإنشاء بوت والحصول على الـ `Token`.
   - تواصل مع `@userinfobot` للحصول على الـ `Chat ID` الخاص بك.
3. **التشغيل**:
   - `npm install`
   - `npm run dev`

## المميزات:
- تسجيل دخول التجار.
- لوحة تحكم لكل تاجر لإدارة المنتجات وإعدادات تيليجرام.
- صفحة متجر فريدة لكل تاجر.
- إرسال بيانات الزبون (الاسم، الهاتف، العنوان) مباشرة إلى هاتف التاجر.
- تحديثات فورية للمنتجات والطلبات.
